# Komari Minimal Theme

A clean and minimal theme for [Komari Monitor](https://github.com/komari-monitor/komari), built with React, TypeScript, and Vite.

## Installation

1. Open the `Releases` page of this repository.
2. Download the latest theme package named `komari-theme-*.zip`.
3. Go to the theme settings page in Komari.
4. Upload the zip package and apply the theme.

## Highlights

- Clean, focused monitoring dashboard layout
- Responsive design for desktop and mobile
- Built-in theme settings for title, accent color, default view, and more
- Shipped as a ready-to-install theme zip package

## Configurable Options

The theme exposes visual settings through `komari-theme.json`, including:

- Title and subtitle
- Accent color
- Default group
- Default view mode
- Offline node visibility
- Load and ping history windows
- Refresh interval
- Footer text
